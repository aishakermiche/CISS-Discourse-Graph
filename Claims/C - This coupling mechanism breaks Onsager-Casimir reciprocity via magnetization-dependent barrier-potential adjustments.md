# C - This coupling mechanism breaks Onsager-Casimir reciprocity via magnetization-dependent barrier-potential adjustments

## Summary

The impurity-wavefunction coupling mechanism in biomolecular/semiconductor CISS generates a magnetization-dependent effective barrier potential that treats spin-up and spin-down electrons asymmetrically. This asymmetry breaks Onsager-Casimir reciprocity — the symmetry that would otherwise forbid net spin polarization in the linear-response regime without an external magnetic field.
